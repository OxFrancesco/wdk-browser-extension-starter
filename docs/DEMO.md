# Demo Script

Use this outline to record the required 2-5 minute demonstration video.

1. Open `chrome://extensions` or `brave://extensions`, enable Developer Mode, and load `.output/chrome-mv3`.
2. Open the WDK Wallet popup.
3. Generate a seed phrase, enter a password with at least 10 characters, and create the encrypted vault.
4. Switch between Bitcoin, Spark, Ethereum, Polygon, Arbitrum, Plasma, and Solana.
5. Add a second account and show that account addresses are derived separately.
6. Copy a receive address and show its QR code.
7. Use the send form with an invalid address to show automatic network validation.
8. Use the QR image picker to import a recipient address from a QR image.
9. Quote or submit a transaction on a funded test wallet/network.
10. Show the transaction history row and explain the background status refresh.
11. Lock and unlock the wallet to demonstrate password/session protection.

For a production submission, record against funded development wallets and owned RPC/indexer endpoints.
